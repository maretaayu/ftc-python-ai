Variabel & Tipe Data

Variabel di Python itu seperti kotak ajaib yang bisa menyimpan berbagai macam barang. Kamu tinggal kasih nama ke kotak itu, dan isinya bisa kamu ganti kapan saja!

Membuat Variabel:

nama = "Jasmine"
umur = 26
jomblo = True

Di Python, kamu tidak perlu mendeklarasikan tipe data terlebih dahulu. Python otomatis mengenali tipe data berdasarkan nilai yang kamu berikan!

Tipe Data Dasar di Python:

  • String (str): Teks - dibuat dengan tanda kutip. Contoh: "Halo" atau 'Halo'
  • Integer (int): Bilangan bulat. Contoh: 42, -7
  • Float: Bilangan desimal. Contoh: 3.14, -0.001
  • Boolean (bool): Nilai kebenaran - True atau False
  • None: Nilai kosong atau tidak ada (mirip NULL di bahasa lain)

Mengetahui tipe data variabel:

nama = "Budi" print(type(nama)) # Akan menampilkan:

Bayangkan variabel seperti label nama pada gelas. Kamu bisa isi gelas itu dengan teh, kopi, atau jus. Isinya bisa berubah-ubah, tapi labelnya tetap!

Tips: Nama variabel di Python tidak boleh dimulai dengan angka dan tidak boleh mengandung spasi. Gunakan huruf kecil dan underscore untuk memisahkan kata (snake_case) adalah konvensi yang umum di Python, contoh: nama_lengkap, tanggal_lahir.

Input/Output & Manipulasi String

Mari belajar bagaimana caranya berkomunikasi dengan program Python kita!

Output - Menampilkan Data:

print("Selamat datang di toko kami!")
nama = "Dina"
print("Halo,", nama, "! Apa kabar?")

Fungsi print() adalah cara paling mudah untuk menampilkan output. Format string (f-string) memudahkan kita menyisipkan variabel dalam teks.

Input - Menerima Data dari Pengguna:

nama = input("Masukkan nama Anda: ") print(f"Halo {nama}, selamat datang!")

Fungsi input() akan menampilkan pesan dan menunggu pengguna mengetik jawaban, kemudian menyimpannya sebagai string.

Manipulasi String:

String di Python seperti kalung manik-manik, di mana setiap karakter adalah manik yang bisa diakses satu per satu. Mari kita pelajari cara-cara memanipulasi string di Python.

1. Concatenation (Penggabungan String)

Di Python, kita bisa menggabungkan string dengan menggunakan operator +:

nama_depan = "Python" nama_belakang = "Programming" nama_lengkap = nama_depan + " " + nama_belakang print(nama_lengkap) # Output: Python Programming >

2. len() - Menghitung Panjang String

Fungsi len() digunakan untuk menghitung jumlah karakter dalam string:

teks = "Python Keren" panjang = len(teks) print(panjang) # Output: 12 >

3. upper() - Mengubah ke Huruf Kapital

Method upper() mengubah semua karakter string menjadi huruf besar:

teks = "Python adalah bahasa pemrograman" kapital = teks.upper() print(kapital) # Output: PYTHON ADALAH BAHASA PEMROGRAMAN >

4. lower() - Mengubah ke Huruf Kecil

Method lower() mengubah semua karakter string menjadi huruf kecil:

teks = "PYTHON ADALAH BAHASA PEMROGRAMAN" kecil = teks.lower() print(kecil) # Output: python adalah bahasa pemrograman >

5. strip() - Menghapus Spasi di Awal dan Akhir

Method strip() menghapus spasi yang tidak diinginkan di awal dan akhir string:

teks = " Python Keren " bersih = teks.strip() print(bersih) # Output: "Python Keren" >

6. replace() - Mengganti Bagian String

Method replace() mengganti bagian teks dalam string:

teks = "Saya suka Java" ganti = teks.replace("Java", "Python") print(ganti) # Output: "Saya suka Python"

7. f-string - String Interpolation

f-string adalah cara modern untuk menyisipkan nilai variabel ke dalam string:

nama = "Budi" umur = 25 pesan = f"Nama saya {nama} dan umur saya {umur} tahun" print(pesan) # Output: "Nama saya Budi dan umur saya 25 tahun"

8. Multiline f-string

f-string juga bisa digunakan untuk teks dengan banyak baris menggunakan triple quotes:

nama = "Ani" umur = 22 hobi = "membaca" biodata = f""" BIODATA ============== Nama : {nama} Umur : {umur} tahun Hobi : {hobi} """ print(biodata)

9. Slicing - Mengambil Sebagian String

Slicing digunakan untuk mengambil sebagian karakter dari string:

teks = "Python Keren" huruf_pertama = teks[0] # Mengambil karakter pertama: 'P' tiga_huruf = teks[0:3] # Mengambil 3 karakter pertama: 'Pyt' terakhir = teks[-1] # Karakter terakhir: 'n'

Logika Kondisional

Logika kondisional adalah cara kita membuat keputusan dalam program. Ini seperti persimpangan jalan - kita memilih jalan yang berbeda berdasarkan situasi.

Pernyataan If-Else:

umur = int(input("Berapa umur kamu? ")) if umur < 17: print("Kamu masih di bawah umur.") print("Belum boleh nonton film dewasa!") elif umur >= 17 and umur < 50: print("Kamu sudah dewasa.") else: print("Kamu sudah senior.") print("Program selesai.") >

Perhatikan indentasi (spasi di awal baris) yang menentukan blok kode. Tanpa kurung kurawal seperti di bahasa lain, Python menggunakan indentasi untuk menetapkan blok kode mana yang menjadi bagian dari kondisi tertentu.

Operator Perbandingan:

  • == : Sama dengan
  • != : Tidak sama dengan
  • < : Kurang dari
  • > : Lebih dari
  • <= : Kurang dari atau sama dengan
  • >= : Lebih dari atau sama dengan

Operator Logika:

is_logged_in = True has_permission = False if is_logged_in and has_permission: print("Anda boleh mengakses fitur ini") elif is_logged_in and not has_permission: print("Anda login tapi tidak punya izin") elif not is_logged_in: print("Silakan login dulu") >

Operator logika memungkinkan kita menggabungkan beberapa kondisi:

  • and : Kedua kondisi harus benar
  • or : Salah satu kondisi benar
  • not : Membalikkan kondisi

Analogi: Kondisi if-elif-else seperti penjaga tiket di bioskop. Pertama dia cek apakah kamu anak-anak (if), remaja (elif), atau dewasa (else), kemudian memberi harga tiket yang sesuai.

Cara Cerdas - Operator Ternary:

umur = 20 status = "Dewasa" if umur >= 17 else "Anak-anak" print(status) # Output: Dewasa

Perulangan

Perulangan (loop) adalah cara kita mengulang tindakan tanpa menulis kode berulang kali. Bayangkan kamu mau memotong 100 keping roti - daripada menuliskan "potong roti" 100 kali, kamu cukup tulis "potong roti 100 kali"!

Loop For:

Loop For biasanya digunakan saat kita tahu berapa kali kita ingin mengulang sesuatu.

# Contoh loop sederhana for i in range(5): print(f"Ini pengulangan ke-{i+1}")

Loop While:

Loop While digunakan saat kita tidak tahu pasti berapa kali perulangan akan terjadi, tapi kita tahu kapan harus berhenti.

hitung = 1 while hitung <= 5: print(f"Hitungan ke-{hitung}") hitung = hitung + 1

Kontrol Perulangan:

break - keluar dari loop:

for i in range(10): print(i) if i == 3: print("Cukup sampai 3!") break # Hanya mencetak 0, 1, 2, 3, "Cukup sampai 3!"

continue - lompat ke iterasi berikutnya:

for i in range(5): if i == 2: print("Lewati angka 2") continue print(i) # Mencetak: 0, 1, "Lewati angka 2", 3, 4

Analogi: Loop For seperti resep masak dengan langkah-langkah yang jelas: "Uleni adonan sebanyak 10 kali". Loop While seperti mengatakan "Uleni adonan sampai terasa kenyal".
